### ***_How to Run:_***

<h4>2. Install python 3.10.6 </h4>

<h4>4. Activate a virtual environment </h4>
<h4>5. Install dependencies in virtual environment </h4>

<h4>7. Run the command "python main.py"/"python3 main.py" to run the app </h4>
